# Movie
Aplikasi Movie yang terintegrasi secara real time dengan api.themoviedb.org

Fitur
1. Search Movie & Tv Show
2. Upcoming Movie
3. Upcoming Tv Show
4. Favorite List
5. Daily Reminder
6. Release Today Reminder
7. Widget Favorite
